# Wireframes - Baja fidelidad

## Que es?

Diseño de baja fidelidad que muestra la estructura sin detalles graficos.

## Propsitos

- Definir layout
- Identificar componentes
- Validar flujo
- Comunicar idea

## Elementos

- Cajas para secciones
- Lineas para texto
- Simbolos para imagenes
- Botones simples

## No incluir

- Colores
- Fuentes
- Detalles graficos
- Animaciones

## Ejemplo

```
┌─────────────────────────────────────┐
│ [Logo]          [Menu] [Login]     │  Header
├─────────────────────────────────────┤
│                                     │
│  ┌─────────┐  ┌─────────┐          │
│  │ Imagen  │  │ Imagen  │          │  Hero
│  └─────────┘  └─────────┘          │
│                                     │
│  Titulo Principal                  │  Content
│  Descripcion del contenido          │
│                                     │
│  [Boton Principal]                │
│                                     │
├─────────────────────────────────────┤
│ Copyright 2026                     │  Footer
└─────────────────────────────────────┘
```

## Herramientas

- Excalidraw
- Balsamiq
- Figma (baja fidelidad)
- Paper/pencil

## Reglas

1. Rapido de crear
2. Facil de modificar
3. Enfocarse en funcion, no forma
4. Iterar muchas veces
5. Validar con usuarios

## Para crear desde descripcion

1. Identificar secciones principales
2. Definir componentes
3. Dibujar layout
4. Añadir interacciones basicas
5. Probar flujo
